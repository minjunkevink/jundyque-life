Welcome to my blog! This is a sample post to get you started.

## Getting Started

This is a minimalistic Jekyll blog template. You can customize it to your liking by:

1. Editing the `_config.yml` file to update site settings
2. Modifying the layouts in `_layouts/` to change the design
3. Adding your own posts in `_posts/` directory
4. Customizing the CSS in `assets/css/style.css`

## Writing Posts

To create a new blog post:

1. Create a new file in the `_posts/` directory
2. Name it with the format: `YYYY-MM-DD-post-title.md`
3. Add front matter at the top with layout, title, date, and tags
4. Write your content in Markdown

## Features

- Clean, minimalistic design
- Responsive layout
- Tag filtering
- Search functionality
- Related posts

Enjoy blogging!
